# <bigfont> 甲</bigfont> - Turtle Shell 

Previous kanji: [系](系.md) Next kanji: [召](召.md) 

## Meanings
 Turtle Shell
## Mnemonics
 * Poor turtles. We don't like them here. This is a <span style="background-color:#ADD8E6"> turtle shell</span>. See the shell with just a tail remaining? It's the only thing we didn't remove off the turtle. It's just a shell with a tail, so basically it's just a turtle shell.
