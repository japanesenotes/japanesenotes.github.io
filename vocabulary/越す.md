## Mnemonics:

* 
* Because the hiragana ends with an <span style="background-color:#fed8b1"> [う](https://jisho.org/search/う)</span> sound, you know this word is a verb. The kanji itself means <span style="background-color:#ffcccb"> go beyond</span> so the verb vocab version is also <span style="background-color:#ffcccb"> to go beyond</span>.
* The reading can come from <span style="background-color:#fed8b1"> [越える](https://jisho.org/search/越える)</span>. So if you know that you know this too!
