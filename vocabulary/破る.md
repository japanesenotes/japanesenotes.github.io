## Mnemonics:

* 
* Because the hiragana ends with an <span style="background-color:#fed8b1"> [う](https://jisho.org/search/う)</span> sound, you know this word is a verb. The kanji itself means <span style="background-color:#ffcccb"> tear</span> so the verb version is <span style="background-color:#ffcccb"> to tear</span>.
* You start <span style="background-color:#ffcccb"> to tear</span> the cloth. You don't know if you're supposed to or not. The crowd isn't sure either. You hear kind of a <span style="background-color:#ffcccb"> yah boo</span> (<span style="background-color:#fed8b1"> [やぶ](https://jisho.org/search/やぶ)</span>) reaction going on, making you unsure if you should continue or stop.
