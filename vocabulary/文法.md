## Mnemonics:
私の上司、見た目はできる男なのに、いつも間違った文法を使うのが残念すぎるよ。
* My boss looks like a smart person, so it’s too bad that he always makes grammar mistakes.
* The <span style="background-color:#ffcccb"> method</span> of <span style="background-color:#ffcccb"> writing</span> is how you write. You can have vocabulary words, but you have to be able to put them together using some kind of method. This method happens to be <span style="background-color:#ffcccb"> grammar</span>.
* This is a jukugo word, which usually means on'yomi readings from the kanji. If you know the readings of your kanji you'll know how to read this as well. The <span style="background-color:#fed8b1"> [ほう](https://jisho.org/search/ほう)</span> changes to <span style="background-color:#fed8b1"> [ぽう](https://jisho.org/search/ぽう)</span>.
