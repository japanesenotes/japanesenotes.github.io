## Mnemonics:

* 
* This is a single kanji with an <span style="background-color:#fed8b1"> [い](https://jisho.org/search/い)</span> on the end, meaning you know it's probably an adjective. What's the adjective form of <span style="background-color:#ffcccb"> dull</span>? It's <span style="background-color:#ffcccb"> dull</span>.
* To remember the reading, think about how when you're i<span style="background-color:#ffcccb"> neb</span>riated (<span style="background-color:#fed8b1"> [にぶ](https://jisho.org/search/にぶ)</span>) you need to use a <span style="background-color:#ffcccb"> dull</span> knife. The <span style="background-color:#fed8b1"> [にぶ](https://jisho.org/search/にぶ)い</span> reading talks about a thing that is dull. For <span style="background-color:#fed8b1"> [のろい]([のろ](https://jisho.org/search/のろ)い)</span>, which refers to a dull mind (someone who's <span style="background-color:#ffcccb"> thickheaded</span>). Think about a <span style="background-color:#ffcccb"> Nor</span>se (<span style="background-color:#fed8b1"> [のろ](https://jisho.org/search/のろ)</span>) God, the God of dullbrains.
