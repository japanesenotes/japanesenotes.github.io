## Mnemonics:
花火ってめっちゃいい匂いだよね？
* Fireworks have a great smell, don’t they?
* The <span style="background-color:#ffcccb"> flower</span> <span style="background-color:#ffcccb"> fire</span>... what looks like flowers on fire? That would be <span style="background-color:#ffcccb"> fireworks</span>. Totally makes sense, right?
* The readings for the kanji that make this word are the kun'yomi readings, where fire's <span style="background-color:#fed8b1"> [ひ](https://jisho.org/search/ひ)</span> changes to <span style="background-color:#fed8b1"> [び](https://jisho.org/search/び)</span>. You already learned the kun'yomi reading for <span style="background-color:#fed8b1"> [花](https://jisho.org/search/花)</span> when you learned the kanji, so it's only a matter of remembering to use the kun'yomi for <span style="background-color:#fed8b1"> [火](https://jisho.org/search/火)</span>. If you do, you should be able to read this word.
* 花, 火
* Flower, Fire
