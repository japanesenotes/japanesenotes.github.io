## Mnemonics:
仏教に伴って、お香は六世紀に日本に伝来しました。
* Along with Buddhism, incense was brought to Japan in the 6th century.
* The <span style="background-color:#ffcccb"> buddha</span> <span style="background-color:#ffcccb"> teach</span>ings are what you have to learn when you're studying <span style="background-color:#ffcccb"> Buddhism</span>.
* This is a jukugo word, which usually means on'yomi readings from the kanji. If you know the readings of your kanji you'll know how to read this as well. Note that <span style="background-color:#fed8b1"> [仏](https://jisho.org/search/仏)</span> gets changed to <span style="background-color:#fed8b1"> [ぶっ](https://jisho.org/search/ぶっ)</span> for this one.
* 仏, 教
* Buddha, Teach
