## Mnemonics:

* 
* If <span style="background-color:#fed8b1"> [催促](https://jisho.org/search/催促)</span> is <span style="background-color:#ffcccb"> urge</span>, then the verb version would be <span style="background-color:#ffcccb"> to urge</span>.
* This is a jukugo word, which usually means on'yomi readings from the kanji. If you know the readings of your kanji you'll know how to read this as well. Will help to know <span style="background-color:#fed8b1"> [催促](https://jisho.org/search/催促)</span> first.
* 催, 促
* Sponsor, Urge
