## Mnemonics:
中小企業は小回りが利くから、こういう仕事をお願いしても平気だろう。
* Smaller companies take on new challenges more rapidly, so they probably won't mind if we request a job like this.
* The noun form of <span style="background-color:#ffcccb"> request</span> (don't be tricked by the <span style="background-color:#fed8b1"> [い](https://jisho.org/search/い)</span> on the end, it's not an adjective). So, request goes to <span style="background-color:#ffcccb"> a request</span>.
* The reading is the same like <span style="background-color:#fed8b1"> [お願いします](https://jisho.org/search/お願いします)</span>. Hopefully you're able to read that by now so you're able to read this too.
