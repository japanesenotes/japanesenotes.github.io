## Mnemonics:

* 
* Because the hiragana ends with an <span style="background-color:#fed8b1"> [う](https://jisho.org/search/う)</span> sound, you know this word is a verb. The kanji itself means <span style="background-color:#ffcccb"> rescue</span> so the verb vocab version is <span style="background-color:#ffcccb"> to rescue</span>.
* The <span style="background-color:#fed8b1"> [すく](https://jisho.org/search/すく)</span> in the reading is a lot like the <span style="background-color:#ffcccb"> scue</span> (<span style="background-color:#fed8b1"> [すく](https://jisho.org/search/すく)う</span>) part of <span style="background-color:#ffcccb"> to rescue</span>, don't you think? Hopefully you do, so you can make the connection.
