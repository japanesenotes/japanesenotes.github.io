## Mnemonics:

* 
* Because the hiragana ends with an <span style="background-color:#fed8b1"> [う](https://jisho.org/search/う)</span> sound, you know this word is a verb. The kanji itself means <span style="background-color:#ffcccb"> belong</span> so the verb vocab version is <span style="background-color:#ffcccb"> to belong to</span>.
* The reading is the one you learned with the kanji. This kanji is going <span style="background-color:#ffcccb"> to belong to</span> you.
