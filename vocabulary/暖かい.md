## Mnemonics:

* 
* This is a single kanji with an <span style="background-color:#fed8b1"> [い](https://jisho.org/search/い)</span> on the end, meaning you know it's probably an adjective. What's the adjective form of <span style="background-color:#ffcccb"> warm</span>? It's <span style="background-color:#ffcccb"> warm</span> as well! Though this is for a <span style="background-color:#ffcccb"> warm</span> weather or temperature, not some warm food or something like that, that would be <span style="background-color:#fed8b1"> [温かい](https://jisho.org/search/温かい)</span>, a different <span style="background-color:#fed8b1"> [あたたかい](https://jisho.org/search/あたたかい)</span>. Confusing, right?
* The reading is the same like <span style="background-color:#fed8b1"> [温かい](https://jisho.org/search/温かい)</span>, which also means <span style="background-color:#ffcccb"> warm</span>, though it refers to a warm thing unlike <span style="background-color:#fed8b1"> [暖かい](https://jisho.org/search/暖かい)</span> which refers to a warm temperature or place.<br />*Note that <span style="background-color:#fed8b1"> [あったかい ](https://jisho.org/search/あったかい )</span> is for spoken Japanese. The correct reading is still <span style="background-color:#fed8b1"> [あたたかい](https://jisho.org/search/あたたかい)</span>. But we've added this reading too for you chattygattors out there.
* 暖
