## Mnemonics:

* 
* A <span style="background-color:#ffcccb"> river</span> of <span style="background-color:#ffcccb"> willow</span>s... so hilarious! What a great <span style="background-color:#ffcccb"> comic haiku</span>, that one is. Try to think of a comic haiku about the river willow, if you can.<br />The River Willow<br />Flows upstream like a salmon<br />This is not funny
* This is a jukugo word that uses the on'yomi readings of the kanji. And for once the kanji <span style="background-color:#fed8b1"> [川](https://jisho.org/search/川)</span> doesn't use the kun'yomi reading. When you think of a <span style="background-color:#ffcccb"> comic haiku</span>, think of something hilarious... like a <span style="background-color:#ffcccb"> cen</span>taur (<span style="background-color:#fed8b1"> [せん](https://jisho.org/search/せん)</span>), doing a standup act with a comedy haiku.
