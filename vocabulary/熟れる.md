## Mnemonics:

* 
* Because the hiragana ends with an <span style="background-color:#fed8b1"> [う](https://jisho.org/search/う)</span> sound, you know this word is a verb. The kanji itself means <span style="background-color:#ffcccb"> ripen</span> so the verb vocab version is <span style="background-color:#ffcccb"> to ripen</span>.
* Since this word consists of a kanji with hiragana attached, you can bet that it will use the kun'yomi reading. You didn't learn that reading with this kanji, so here's a mnemonic to help you: If something starts <span style="background-color:#ffcccb"> to ripen</span> too much, it gets all gross, and you're all like, <span style="background-color:#ffcccb"> ew</span> (<span style="background-color:#fed8b1"> [う](https://jisho.org/search/う)</span>), that's gross.
