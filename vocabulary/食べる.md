## Mnemonics:
私は、寝ながら食べるのが好きだ。夢遊病のような感じで、それは夢遊食事病と呼ばれる。
* I like to eat while I’m sleeping. It’s just like sleepwalking, but it’s called sleep-eating.
* This word consists of kanji with hiragana attached. Because the hiragana ends with an <span style="background-color:#fed8b1"> [う](https://jisho.org/search/う)</span> sound, you know this word is a verb. The kanji itself means <span style="background-color:#ffcccb"> eat</span> so the verb version is <span style="background-color:#ffcccb"> to eat</span>.
* You have to remember the <span style="background-color:#fed8b1"> [た](https://jisho.org/search/た)</span> portion if you want to be able to read this word, which uses the kun'yomi reading. Think about yourself <span style="background-color:#ffcccb"> eat</span>ing some <span style="background-color:#ffcccb"> ta</span>cos (<span style="background-color:#fed8b1"> [た](https://jisho.org/search/た)</span>). Imagine yourself eating it, that raw beef taste in your mouth (either like it or hate it).
