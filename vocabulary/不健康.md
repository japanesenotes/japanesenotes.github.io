## Mnemonics:

* 
* If <span style="background-color:#fed8b1"> [健康](https://jisho.org/search/健康)</span> is health, then something that is <span style="background-color:#ffcccb"> not</span> this is <span style="background-color:#ffcccb"> unhealthy</span>.
* This is a jukugo word, which usually means on'yomi readings from the kanji. If you know the readings of your kanji you'll know how to read this as well. Will help to know <span style="background-color:#fed8b1"> [健康](https://jisho.org/search/健康)</span> first.
