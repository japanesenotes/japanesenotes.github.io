## Mnemonics:

* 
* Because the hiragana ends with an <span style="background-color:#fed8b1"> [う](https://jisho.org/search/う)</span> sound, you know this word is a verb. The kanji itself means <span style="background-color:#ffcccb"> fear</span> so the verb version is <span style="background-color:#ffcccb"> to fear</span>.
* The reading is shared with <span style="background-color:#fed8b1"> [恐ろしい](https://jisho.org/search/恐ろしい)</span>, which is fear. Use that, and don't be afraid. You'll be able to learn this reading yet.
