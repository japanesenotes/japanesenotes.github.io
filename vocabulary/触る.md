## Mnemonics:

* 
* Because the hiragana ends with an <span style="background-color:#fed8b1"> [う](https://jisho.org/search/う)</span> sound, you know this word is a verb. The kanji itself means <span style="background-color:#ffcccb"> touch</span> so the verb vocab version is <span style="background-color:#ffcccb"> to touch</span>.
* Since this word consists of a kanji with hiragana attached, you can bet that it will use the kun'yomi reading. You didn't learn that reading with this kanji, so here's a mnemonic to help you: You're an idiot. You try <span style="background-color:#ffcccb"> to touch</span> a <span style="background-color:#ffcccb"> saw</span> (<span style="background-color:#fed8b1"> [さわ](https://jisho.org/search/さわ)</span>).
