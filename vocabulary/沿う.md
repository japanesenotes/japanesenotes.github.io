## Mnemonics:

* 
* Because the hiragana ends with an <span style="background-color:#fed8b1"> [う](https://jisho.org/search/う)</span> sound, you know this word is a verb. The kanji itself means <span style="background-color:#ffcccb"> run alongside</span> so the verb vocab version is <span style="background-color:#ffcccb"> to run alongside</span>.
* Think about how you're <span style="background-color:#ffcccb"> sew</span>ing (<span style="background-color:#fed8b1"> [そう](https://jisho.org/search/そう)</span>) something, and your thread has <span style="background-color:#ffcccb"> to run alongside</span> the edge of the cloth.
