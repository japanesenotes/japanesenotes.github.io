## Mnemonics:
そう悲しむなって。お前の日本語は今はそれで十分だよ。ただ、勉強することを投げ出さずに、少しずつ上手くなればいいさ。
* Don’t be sad. Your Japanese is good enough for now. Just don’t stop learning and you’ll improve little by little.
* This word consists of kanji with hiragana attached. Because the hiragana ends with an <span style="background-color:#fed8b1"> [う](https://jisho.org/search/う)</span> sound, you know this word is a verb. The kanji itself means <span style="background-color:#ffcccb"> sad</span> so the verb form of that is <span style="background-color:#ffcccb"> to be sad</span>. Boohoo.
* If you're going <span style="background-color:#ffcccb"> to be sad</span> about something right now, it will be <span style="background-color:#ffcccb"> kana</span> (<span style="background-color:#fed8b1"> [かな](https://jisho.org/search/かな)</span>). Don't use kana when you're trying to learn kanji! You'll be sad forever if you're stuck using kana your whole life, so don't let that habit start now, friend.
