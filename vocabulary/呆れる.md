## Mnemonics:

* 
* This word consists of kanji with hiragana attached. Because the hiragana ends with an <span style="background-color:#fed8b1"> [う](https://jisho.org/search/う)</span> sound, you know this word is a verb. The kanji itself means <span style="background-color:#ffcccb"> shock</span>, so the verb version is <span style="background-color:#ffcccb"> to be shocked</span> or <span style="background-color:#ffcccb"> to be amazed</span>.
* Since this word consists of a kanji with hiragana attached, you can bet that it will use the kun'yomi reading. You didn't learn that reading with this kanji, so here's a mnemonic to help you: <br />If you were going <span style="background-color:#ffcccb"> to be shocked</span>, you'd shout, <span style="background-color:#ffcccb"> ACK</span> (<span style="background-color:#fed8b1"> [あき](https://jisho.org/search/あき)</span>)!
