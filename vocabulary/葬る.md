## Mnemonics:

* 
* Because the hiragana ends with an <span style="background-color:#fed8b1"> [う](https://jisho.org/search/う)</span> sound, you know this word is a verb. The kanji itself means <span style="background-color:#ffcccb"> burial</span>, so the verb version would be <span style="background-color:#ffcccb"> to bury</span> or <span style="background-color:#ffcccb"> to inter</span>.
* We're going <span style="background-color:#ffcccb"> to bury</span> you, so this is your new <span style="background-color:#ffcccb"> home</span> (<span style="background-color:#fed8b1"> [ほうむ](https://jisho.org/search/ほうむ)</span>) now, friend.
