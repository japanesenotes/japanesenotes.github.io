## Mnemonics:

* 
* Because the hiragana ends with an <span style="background-color:#fed8b1"> [う](https://jisho.org/search/う)</span> sound, you know this word is a verb. The kanji itself means <span style="background-color:#ffcccb"> divide</span> so the verb vocab version is <span style="background-color:#ffcccb"> to divide</span>.
* The reading you learned with the kanji is the one you'll use here. It's the <span style="background-color:#fed8b1"> [わ](https://jisho.org/search/わ)</span> one, though, making it <span style="background-color:#fed8b1"> [わ](https://jisho.org/search/わ)る</span>.
