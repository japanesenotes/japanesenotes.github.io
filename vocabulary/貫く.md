## Mnemonics:

* 
* This word consists of kanji with hiragana attached. Because the hiragana ends with an <span style="background-color:#fed8b1"> [う](https://jisho.org/search/う)</span> sound, you know this word is a verb. The kanji itself means <span style="background-color:#ffcccb"> pierce</span>, so the verb version is <span style="background-color:#ffcccb"> to go through</span> or <span style="background-color:#ffcccb"> to pierce</span>.
* Since this word consists of a kanji with hiragana attached, you can bet that it will use the kun'yomi reading. You didn't learn that reading with this kanji, so here's a mnemonic to help you: <br />The best way <span style="background-color:#ffcccb"> to go through</span> someones mind or a college campus is <span style="background-color:#ffcccb"> to run nude</span> (<span style="background-color:#fed8b1"> [つらぬ](https://jisho.org/search/つらぬ)</span>). So if you really want <span style="background-color:#ffcccb"> to pierce</span> someone's mind and soul remember this!
