## Mnemonics:
オタクだけではなく他の人にとっても、ネット上で大文字を使うことは大声を出すことに等しいですよ。
* Even for people who aren’t nerds, using uppercase letters on the internet is equivalent to shouting.
* This is a single kanji with an <span style="background-color:#fed8b1"> [い](https://jisho.org/search/い)</span> on the end, meaning you know it's probably an adjective. What's the adjective form of <span style="background-color:#ffcccb"> equal</span>? It's <span style="background-color:#ffcccb"> equal</span> (and also <span style="background-color:#ffcccb"> similar</span> or <span style="background-color:#ffcccb"> equivalent</span>). That similar one is a bit weird, because similar is not equal, though I'd lean more towards equal.
* Since this word consists of a kanji with hiragana attached, you can bet that it will use the kun'yomi reading. You didn't learn that reading with this kanji, so here's a mnemonic to help you: What should be equal above all things? I think each person (<span style="background-color:#ffcccb"> ひと</span>) should be equal!
* 等
