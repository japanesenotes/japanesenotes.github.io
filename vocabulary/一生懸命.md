## Mnemonics:

* 
* You have only <span style="background-color:#ffcccb"> one</span> <span style="background-color:#ffcccb"> life</span> so do everything with lots of <span style="background-color:#ffcccb"> eager</span>ness (<span style="background-color:#fed8b1"> [懸命](https://jisho.org/search/懸命)</span>). Do everything <span style="background-color:#ffcccb"> with all one's might</span>. Go <span style="background-color:#ffcccb"> all out</span>.
* Everything except the <span style="background-color:#fed8b1"> [生](https://jisho.org/search/生)</span> uses the readings you learned with the kanji. To remember <span style="background-color:#fed8b1"> [生](https://jisho.org/search/生)</span> as <span style="background-color:#fed8b1"> [しょう](https://jisho.org/search/しょう)</span>, think of how you are running your own Broadway <span style="background-color:#ffcccb"> show</span> (<span style="background-color:#fed8b1"> [しょう](https://jisho.org/search/しょう)</span>), and you're doing it <span style="background-color:#ffcccb"> with all one's might</span> (your might). You're going <span style="background-color:#ffcccb"> all out</span>.
