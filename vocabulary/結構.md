## Mnemonics:

* 
* <span style="background-color:#ffcccb"> Bind</span> and <span style="background-color:#ffcccb"> set up</span> everything so it is <span style="background-color:#ffcccb"> fairly</span> <span style="background-color:#ffcccb"> fine</span> and <span style="background-color:#ffcccb"> alright</span>. This word is used like this:<br />That's fine, no more <span style="background-color:#fed8b1"> [結構です](https://jisho.org/search/結構です)</span><br />It's fairly big <span style="background-color:#fed8b1"> [結構大きい](https://jisho.org/search/結構大きい)</span><br />It's alright <span style="background-color:#fed8b1"> [結構です](https://jisho.org/search/結構です)</span><br />etc.
* This is a jukugo word, which usually means on'yomi readings from the kanji. If you know the readings of your kanji you'll know how to read this as well. The <span style="background-color:#fed8b1"> [けつ](https://jisho.org/search/けつ)</span> shortens to <span style="background-color:#fed8b1"> [けっ](https://jisho.org/search/けっ)</span>.
* 結, 構
* Bind, Set Up
