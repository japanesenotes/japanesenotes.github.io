## Mnemonics
 * The <span style="background-color:#ADD8E6"> leader</span> of <span style="background-color:#ADD8E6"> righteousness</span> is the only one who can run this <span style="background-color:#ffcccb"> ceremony</span>. Why? Because it's a very pure ceremony, and only those with an extremely righteous mind can run it.
* Have him run your ceremony for you. Think about why he is so righteous, then think about an extremely righteous ceremony.
* Sadly, the reason only the righteous can run this <span style="background-color:#ffcccb"> ceremony</span> is because it's a punishment ceremony, where he pulls the cord to cut someone's head off via a <span style="background-color:#ffcccb"> gui</span>llotine (<span style="background-color:#fed8b1"> [ぎ](https://jisho.org/search/ぎ)</span>).
* *CHOP*
